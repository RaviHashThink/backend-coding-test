Rides

Ride - Get ride information

API

GET
/rides/:id

Parameters

Field Type Description Mandatory
id Number Rides unqiue id Yes

Response: Success 200 OK

[
    {
        "rideID": 1,
        "startLat": 1.22334,
        "startLong": 2.33434,
        "endLat": 4.8877,
        "endLong": 8.00999,
        "riderName": "ravi",
        "driverName": "teja",
        "driverVehicle": "polo",
        "created": "2019-08-06 13:21:16"
    }
]

Response: ERROR

{
    error_code: 'SERVER_ERROR',
    message: 'Unknown error'
}

Code Message Description
SERVER_ERROR Unknown error Server error when trying to fetch the ride information
RIDES_NOT_FOUND_ERROR Could not find any rides Ride information is not exist for the given id


Rides - Get all rides information

API

GET
/rides?limit=10&page=1&sort=created&order=asc&fetch_total_count=true

Parameters

Field Type Description Mandatory
limit Number No of records per page. Default all Optional
page Number Pagination page number Optional
sort String Name of field to be sort. Default "created" Optional
order String Sorting order (asc / desc). Default "asc" Optional
fetch_total_count Number To get the count of total number of records Optional

Response: Success 200 OK

{
      totalCount: 20,
      records : [
          {
              "rideID": 1,
              "startLat": 1.22334,
              "startLong": 2.33434,
              "endLat": 4.8877,
              "endLong": 8.00999,
              "riderName": "ravi",
              "driverName": "teja",
              "driverVehicle": "polo",
              "created": "2019-08-06 13:21:16"
          }
      ]
}

Response: ERROR

{
    error_code: 'SERVER_ERROR',
    message: 'Unknown error'
}

Code Message Description
SERVER_ERROR Unknown error Server error when trying to fetch the rides information
RIDES_NOT_FOUND_ERROR Could not find any rides No rides information available
